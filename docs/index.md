# pypath

Molecular signaling prior knowledge processing.

pypath is a Python library for building molecular interaction databases
from various resources. It serves as the data integration backend for
[OmniPath](https://omnipathdb.org).

## Quick start

```bash
pip install pypath-omnipath
```

```python
import pypath
```
